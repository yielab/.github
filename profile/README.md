### [Tack](https://github.com/yielab/tack)
Self-hosted project manager in a single binary.

One `~5 MB` statically-linked binary runs the entire stack — GUI, API, and SQLite engine. No Docker, no database server. Kanban, Gantt timeline, sprints, calendar, FTS, REST API + CLI with `--json`, GitHub and Linear import, webhooks, and cross-domain vocabulary (software sprint → kitchen renovation → thesis, same tool). Cross-platform release binaries.

Ops layer for OpenClaw agent fleets.

### [Adapta](https://github.com/yielab/adapta)
Private LLM customization on your own hardware.

RAG (cited retrieval, CPU-only) and LoRA fine-tuning (QLoRA, GPU) behind one OpenAI-compatible endpoint. An eval gate blocks any adapter that doesn't clear a held-out threshold — you can't accidentally deploy a regression. Includes vision fine-tuning (VLMs for invoice extraction, visual QC). No telemetry. Your data stays on your hardware.

## How we build

| Practice | What it means |
|----------|--------------|
| **Spec before code** | Every command and endpoint has a written spec (RFC 2119 keywords) before implementation starts |
| **Honest status tables** | READMEs say what works, what's in progress, and what isn't built yet |
| **Real test coverage** | Unit, integration, contract (schemathesis), e2e (Playwright), and evals where relevant |
| **CI on every push** | Nothing merges without a green pipeline |
| **No inflated claims** | Metrics are generated by the code, not written by hand |

All projects are in active development. `good first issue` labels mark safe entry points for contributors.
